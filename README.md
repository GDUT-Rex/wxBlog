# 微信小程序

## 介绍

这是一款后台为 `python-flask` 的博客微信小程序。使用了 `ColorUI` 组件库。

可以实现 `代码高亮` `点赞` `收藏` `评论` `回复` `接收通知` 等功能。

微信搜索 ： `jarry007` 进行体验。
